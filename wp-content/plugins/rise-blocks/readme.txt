== Description ==

Rise Blocks is a gallery of light weight block for Gutenberg editor. With rise block, you can make stunning layouts for your website or blog with zero knowledge in coding.

https://www.youtube.com/watch?v=Hwljubang2M

Watch beautiful [Live Readymade Demos](https://wpactivethemes.com/live-preview/) to learn what you can do with these blocks.

The main reason behind developing Rise Blocks is to deliver awesome layouts through light weight blocks. This way, we can keep our website light and free from unwanted codes. 

So if you are also inclined to use light weights then Rise Blocks is for you.
Currently we have a couple of great blocks that will help you to create an awesome website. Additionally, you can also check out our WordPress, [Gutenberg Friendly Themes](https://wpactivethemes.com/downloads/category/premium-wordpress-themes/), based in Rise Blocks.

We have used self hosted service from https://eaglevisionit.com to import blocks templates.

### Why Rise Block?

The blocks are light weight which will help to create an awesome websites free from unwanted codes. Its very easy to use, even with zero knowledge in coding can build a stunning website.

There’s all you need to build from simple to complex websites.

### Rise Blocks

**Blog**

This block allows to display five blog articles’s experts and metadata in your desired page. You can fetch it either specific category or all posts and then display it according to publishing date or alphabetical order of the title.

**Call to action**

This block helps to make attractive call to action section with customized background image and button. You can further personalize it through customizing heading with your choice of fonts, and colors.

**Counter**

This block allows to display the numbers in most peculiar way than other content. Additionally you can customize the blocks through tweaking the colors and typography.

**Advanced Heading**

This block enables attractive heading for the page or section with the label feature to show extra information about the topic. You can also customize it further through tweaking settings available in block options in appearance, typography, and color areas.

**Section**

This block enables you to add a section with a desired layout that allows you to add other block within it.

**Page Slider**

This block allows you to fetch other pages’ heading and excerpts; and display it in a carousel layout with a link. You can customize the appearance and also tweak colors.

**Profile Cards**

Add this block to share your energetic team members. You can add their photo, name, description and social profiles.

**Icon Boxes**

This block enables you to display information along with Icon and show it either in grid or in a row.

**Advanced Button**

Modern and flexible buttons for gutenberg.

**Social Icons**

This block enables you to display social icons and set a link to each icon to the relevant social profile with different design options.
 
**Accordion**

This block allows you to add collapsible content/faqs on a page or post. It helps for reducing scrolling and categorizing information.

**News Block #1**

This block enables blog articles either all posts or by category to create a grid layout for Magazine or Blogging websites in a few seconds.

**Carousel Post**

Carousel Post Block is for creating beautiful Gutenberg post grid blocks, post slider blocks and post carousel blocks quickly.

**Icon List**

This blocks enables you to effortlessly present and style information with icons in a structured list format.

**More BLocks**

We have added most useful blocks in our gallery but this is not the end. Still more to add up in Rise Blocks. So stay tuned.

== Installation ==

1. Upload the `rise-blocks` folder to your `/wp-content/plugins/` directory or alternatively upload the rise-blocks.x.x.x.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
2. Activate the Rise Blocks WordPress plugin through the 'Plugins' menu in WordPress.
4. Use Rise Blocks on your next page or post.

== Frequently Asked Questions ==

= Why should we use Rise Blocks? =

Rise Blocks is a minimalists plugin that can build beautiful pages. It's very easy to use, even with zero knowledge in coding can build a stunning website.

= Is there any requirement to use rise blocks on our website? =

The requirement for Rise Block to run smoothly is the latest version of WordPress (above version 5) with Gutenberg Page Editor enabled.

= Do the Rise Blocks have a negative impact on my website’s page load speed? =

Definitely Not. As we have developed Rise Block focusing on lightweights, and clean code. Therefore it won’t affect your web page’s load speed.
